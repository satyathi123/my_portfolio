# Anand Satyarthi - Professional Portfolio

A modern, high-performance, and visually striking personal portfolio website built with React and Vite. It showcases my skills, work experience, featured projects, and educational background through an engaging, responsive "Midnight Mesh" aesthetic.

## 🚀 Live Demo
*(Link to live deployment - Coming soon)*

## ✨ Key Features
- **Glassmorphic UI:** Beautiful semi-transparent frosted styling with deep slate/navy backgrounds.
- **Ambient Mesh Gradients:** CSS-driven animated background floating orbs that interact dynamically with the glass panels.
- **Single-Page Application:** Sleek transitions and instant navigation using a fixed, blurred frosted-glass navbar.
- **Fully Responsive:** Custom media queries ensuring perfect viewing layout across mobile, tablet, and desktop devices.
- **Interactive Micro-animations:** Engaging hover states, smooth scaling, and robust drop-shadows across skill chips and project cards.

## 🛠️ Technology Stack
- **Core:** [React 18](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Architecture:** Component-driven development architecture.
- **Styling:** Custom Vanilla CSS with modern CSS Variables (Design Tokens) and CSS Grid/Flexbox layouts.
- **Iconography:** [Lucide React](https://lucide.dev/) & Inline SVGs.

## 💻 Local Development

Follow these simple steps to run the portfolio locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/satyathi123/my_portfolio.git
   cd my_portfolio
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Start the Vite development server:**
   ```bash
   npm run dev
   ```

4. **View in browser:**
   Open [http://localhost:5175](http://localhost:5175) to visualize the site.
